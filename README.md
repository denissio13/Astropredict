# 🌌 AstroPredict - Астрологический Анализ Крипторынка с GPU Ускорением

Инструмент для прогнозирования движения криптовалютных пар на основе астрологических паттернов (фазы Луны, положения планет, аспекты) с использованием исторических данных. Поддерживает автоматическую загрузку данных с Binance и Bybit, бэктестинг стратегий и оптимизацию параметров с ускорением на NVIDIA GPU.

## ✨ Возможности

- **Автоматическая загрузка данных**: Скачивает дневные свечи с Binance или Bybit
- **Астрологические расчеты**: Фаза Луны, знак Зодиака, освещенность, аспекты Солнце-Луна, положения Венеры и Марса
- **Поиск паттернов**: Находит исторические периоды со схожей астрологической сигнатурой
- **Прогноз направления**: Предсказывает бычье/медвежье движение следующей свечи
- **Бэктестинг**: Тестирует стратегию на исторических данных с детальной статистикой
- **Оптимизация**: Автоматический подбор оптимальных параметров и весов факторов
- **GPU Ускорение**: Поддержка NVIDIA CUDA (RTX 3090 и другие) для быстрого перебора
- **Детальная аналитика**: PnL, Profit Factor, Max Drawdown, анализ ошибок и рекомендации

## 🚀 Быстрый старт

### Простой прогноз
```bash
python3 astro_trading_analyzer.py --pairs BTCUSDT
```

### Прогноз для нескольких пар
```bash
python3 astro_trading_analyzer.py --pairs BTCUSDT,ETHUSDT,SOLUSDT --exchange binance
```

### Прогноз на конкретную дату
```bash
python3 astro_trading_analyzer.py --pairs BTCUSDT --target-date 2024-06-15
```

## 🔧 Параметры командной строки

| Параметр | Краткий | Описание | Пример |
|----------|---------|----------|--------|
| `--pairs` | | Список пар через запятую (обязательно) | `BTCUSDT,ETHUSDT` |
| `--exchange` | | Биржа: `binance` или `bybit` | `--exchange bybit` |
| `--target-date` | `-d` | Дата прогноза (YYYY-MM-DD) | `-d 2024-05-20` |
| `--backtest` | | Запустить бэктест на истории | |
| `--optimize` | | Запустить оптимизацию параметров | |
| `--mode` | | Режим оптимизации: `basic` или `full` | `--mode full` |
| `--candles` | | Количество свечей для теста | `--candles 200` |
| `--similarity` | | Порог схожести паттернов | `--similarity 0.85` |
| `--max-matches` | | Максимум совпадений для анализа | `--max-matches 10` |

- **Результаты бэктеста**: `backtest_result.json`

## 🧠 Как это работает

1. **Загрузка данных**: Скрипт скачивает дневные свечи выбранной пары с биржи
2. **Астрологический расчет**: Для каждой даты вычисляются:
   - Фаза Луны (0-1)
   - Знак Зодиака Луны (0-11)
   - Освещенность Луны (%)
   - Аспект между Солнцем и Луной (0°, 60°, 90°, 120°, 180°)
   - Знаки Венеры и Марса
3. **Поиск паттернов**: Сравнивается астрологическая сигнатура последних N дней с историей
4. **Анализ совпадений**: Если найдено достаточно похожих периодов (с учетом весов), анализируется направление движения цены в следующие дни
5. **Прогноз**: На основе статистики прошлых совпадений делается вывод о вероятном направлении

## ⚙️ Веса астрологических факторов (по умолчанию)

| Фактор | Вес | Описание |
|--------|-----|----------|
| Фаза Луны | 3.0 | Наиболее значимый фактор |
| Знак Луны | 2.0 | Положение Луны в Зодиаке |
| Освещенность | 2.0 | Процент видимой части Луны |
| Аспект Солнце-Луна | 2.0 | Мажорные аспекты |
| Знак Венеры | 1.0 | Влияние на настроение рынка |
| Знак Марса | 1.0 | Агрессия/активность |

В режиме `--mode full` скрипт тестирует различные комбинации весов для поиска оптимальной конфигурации.

## 🎯 Рекомендации по использованию

- **Для начала**: Используйте `--mode basic` для быстрой оценки
- **Для тонкой настройки**: Запустите `--mode full` на больших данных (300+ свечей)
- **Оптимальный диапазон**: 200-500 свечей для баланса скорости и точности
- **Минимум сделок**: Ищите конфигурации с 20+ сделками в бэктесте
- **Profit Factor**: Цель > 1.5 для надежной стратегии
- **Drawdown**: Не должен превышать 50% от общей прибыли
